AGR is a modern cloud-based restaurant ordering platform designed for cafés, restaurants, food trucks, and multi-branch businesses.

The platform combines a beautiful customer ordering experience with a real-time kitchen management dashboard, allowing restaurants to manage incoming orders efficiently while providing customers with a fast, intuitive, and responsive ordering experience.

---

# ✨ Features

## 🍽️ Digital Menu

- Beautiful product catalog
- Categories & filters
- Product search
- Responsive layout
- Product availability
- Rich product descriptions
- Modern card-based UI

---

## ⚙️ Product Customization

Every product can include unlimited customization options.

Examples:

- Bread selection
- Milk type
- Sugar level
- Sauces
- Toppings
- Extras
- Product variations
- Kitchen notes

Prices are updated automatically in real time.

---

## 🛒 Smart Shopping Cart

- Live cart updates
- Quantity management
- Remove items
- Order summary
- Dynamic pricing
- Real-time calculations

---

## 🚚 Flexible Ordering

Customers can choose between:

- Pickup
- Delivery
- Future Orders
- Delivery notes
- Coupon support

---

## 🎁 Coupons & Promotions

- Discount coupons
- Percentage discounts
- Fixed amount discounts
- Promotional campaigns
- Copy coupon code
- Automatic validation

---

## 📡 Live Kitchen Dashboard

Restaurant staff receive orders instantly through a live operational dashboard.

### Dashboard includes

- Live incoming orders
- Waiting for pickup
- Order history
- Order status management
- Automatic refresh
- Customer information
- Order details
- Payment summary

---

## 🏪 Branch Information

Each restaurant branch includes:

- Address
- Phone number
- Opening hours
- Waze navigation
- Branch description

- Fully Responsive
- Desktop Support
- Tablet Support
- Mobile Support

- Modern UI
